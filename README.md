# truffle_Adoption_Dapp
Truffle 프레임워크 에서 스마트컨트랙트 구현

펫 입양 프론트라이브러리 + Truffle + Ganache + solidity + nodejs 


## Running the tests

사전에 메타마스크 설치완료후 Ganache 와 MNEMONIC 을 맞춰서 이더리움연결확인

터미널에서 시작

```
npm run dev
```
